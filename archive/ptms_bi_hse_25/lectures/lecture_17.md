## Test-score: два эквивалентных подхода

\begin{itemize}
    \item В предыдущей лекции мы сравнивали наблюдаемое значение (например, $\bar{x}$ или $\tilde{p}$) с границей принятия решения $K$ в исходных единицах измерения.
    \item Альтернативный подход: преобразовать и наблюдаемое значение, и границу решения в стандартизованную шкалу.
    \item \textbf{Ключевая идея}: стандартизация — это монотонное преобразование, которое сохраняет отношения "больше/меньше" между числами.
    \item Тип score зависит от распределения:
    \begin{itemize}
        \item $z_{\text{score}}$ — когда используем стандартное нормальное распределение (тестирование математических ожиданий при известной дисперсии или тест долей признаков)
        \item $t_{\text{score}}$ — когда используем $t$-распределение (неизвестная дисперсия)
    \end{itemize}
    \item Если наблюдаемое значение $> K$ в исходных единицах, то $\text{score} > \text{критическое значение}$ в стандартизованной шкале.
    \item Если наблюдаемое значение $< K$ в исходных единицах, то $\text{score} < \text{критическое значение}$ в стандартизованной шкале.
\end{itemize}

## Test-score: преимущества подхода

\begin{itemize}
    \item \textbf{Универсальность}: одна и та же критическая точка ($z_{\alpha}$ или $t_{df, \alpha}$) используется для всех тестов с соответствующим распределением.
    \item \textbf{Удобство}: не нужно вычислять границу решения $K$ для каждого конкретного случая.
    \item \textbf{Эквивалентность}: оба подхода (сравнение в исходных единицах и сравнение score) дают одинаковый результат, так как стандартизация — монотонное преобразование.
    \item \textbf{Гибкость}: подход работает для разных типов тестов (односторонние, двусторонние) и разных распределений.
\end{itemize}

## p-value: концепция

\begin{itemize}
    \item \textbf{p-value} (p-значение) — это вероятность получить наблюдаемое значение статистики или более экстремальное значение \textit{при условии, что нулевая гипотеза верна}.
    \item p-value показывает, насколько "необычны" полученные нами данные при предполагамой верной нулевой гипотезе.
    \item \textbf{Правило решения через p-value}: Отклонить $H_0$, если $p\text{-value} < \alpha$.
    \item Чем меньше p-value, тем сильнее свидетельство против нулевой гипотезы.
    \item p-value — это альтернативный способ принятия решения, эквивалентный сравнению score с критическим значением.
\end{itemize}

## p-value: пример
### Пример: Анализ длины шеи жирафа

Исследовательская команда изучает популяции жирафов. Исследователи верят, что средняя длина шеи жирафов огромная — около 20 метров. Однако в выборке было получено выборочное среднее 2.8 метров. Команда хочет проверить, действительно ли средняя длина шеи меньше того значения, в которое они верят.
\begin{itemize}
    \item Случайная величина для анализа: $X$ — длина шеи. Дисперсию $X$ предположим известной и правдивой.
    \item Предполагается, что длины шеи следуют нормальному распределению.
\end{itemize}
Данные:
\begin{itemize}
    \item Выборка: $n = 20$ жирафов
    \item Выборочное среднее: $\bar{x} = 2.8$ метров
    \item Известная дисперсия $X$: $\text{Var }[X] = 0.16$.
\end{itemize}
Гипотезы:
\begin{itemize}
    \item $H_0: \mu = 20$
    \item $H_1: \mu < 20$
\end{itemize}

---

## p-value: решение через score

\begin{itemize}
    \item \textbf{Гипотезы}: $H_0: \mu = 20$ против $H_1: \mu < 20$ (левосторонний тест)
    
    \item \textbf{Данные}: $\mu_0 = 20$, $\sigma = 0.4$, $n = 20$, $\bar{x} = 2.8$, $\alpha = 0.05$
    
    \item \textbf{$z$-score}:
    $$
    z_{\text{score}} = \frac{\bar{x} - \mu_0}{\sigma/\sqrt{n}} = \frac{2.8 - 20}{0.4/\sqrt{20}} = \frac{-17.2}{0.0894} \approx -192.4
    $$
    
    \item \textbf{Критическое значение}: $z_{0.05} = 1.645$
    
    \item \textbf{Решение через score}: $z_{\text{score}} = -192.4 < -z_{0.05} = -1.645$, поэтому отклоняем $H_0$.
\end{itemize}

---

## p-value: вычисление и интерпретация

\begin{itemize}
    \item \textbf{p-value} для левостороннего теста:
    $$
    p\text{-value} = P_{H_0}(Z < z_{\text{score}}) = P(Z < -192.4) \approx 0
    $$
    
    \item p-value практически равна нулю, что означает крайне малую вероятность получить такое экстремальное значение при верной нулевой гипотезе.
    
    \item \textbf{Решение через p-value}: $p\text{-value} \approx 0 < \alpha = 0.05$, поэтому отклоняем $H_0$.
    
    \item \textbf{Вывод}: Средняя длина шеи жирафов статистически значимо меньше 20 метров. Наблюдаемое значение 2.8 метра крайне противоречит гипотезе о том, что средняя длина равна 20 метрам.
    
    \item \textbf{Эквивалентность}: Оба подхода (сравнение $z_{\text{score}}$ с $z_{\alpha}$ и сравнение $p\text{-value}$ с $\alpha$) дают одинаковый результат.
\end{itemize}
