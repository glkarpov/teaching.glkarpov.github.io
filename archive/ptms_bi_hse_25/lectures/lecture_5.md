## Характеристики С.В.: Дисперсия

- Дисперсия случайной величины - это число, **константа**, которое помогает нам понять степень разброса потенциальных значений случайной величины относительно среднего.

- :::{.callout-definition}
Для случайной величины $X$ её дисперсия обозначается $Var[X]$ (Variance) и определяется как:
$$
    Var[X] =  E\left[ \left( X - E[X] \right)^2 \right]
$$
:::

- Формально говоря, дисперсия это математическое ожидание новой случайной величины:
$$
    Y = g(X) = \left( X - E[X]\right)^2,
$$
описывающей квадрат расстояния между значением случайной величины $X$ и её математическим ожиданием.

- Также вводится понятие стандартного отклонения (standard deviation, std) величины $X$, которое является квадратным корнем из дисперсии: $std[X] = \sqrt{Var[X]}$.

## Характеристики С.В.: Дисперсия

::: {.callout-tip title="Вычисление дисперсии"}
Существует упрощенная формула:
\begin{gather*}
    E\left[ \left( X - E[X] \right)^2 \right] = E\left[  X^2 - 2XE[X] + (E[X])^2  \right] = \\
    = E[X^2] - E\left[2 E[X] X \right] + E \left[ (E[X])^2 \right] = E[X^2] - 2 E[X] E[X] + (E[X])^2 = \\  \boxed{E[X^2] - (E[X])^2}
\end{gather*}

В процессе мы используем свойство линейности математического ожидания, а также знание того, что $E[X]$ - это константа, при этом $E[c] = c$, то есть $E \left[  E[X] \right] = E[X]$.
:::

## Свойства дисперсии
\begin{enumerate}
    \item \textbf{Неотрицательность}: Для любой случайной величины $X$:
    $$Var[X] \geq 0$$

    \item \textbf{Дисперсия константы}: Если $c$ - константа, то:
    $$Var[c] = 0$$

    \item \textbf{Масштабирование}: Для любой константы $c$:
    $$Var[cX] = c^2 Var[X]$$
\end{enumerate}

## Дисперсия: пример 1
Посчитаем дисперсию на примере нашей задачи. Считаем уже известным, что $E[X] = 1.2$.

\begin{center}
    \begin{tabular}{c | c c c c c c} 
        \hline
        $x$ & 0 & -1 & -2 & 1 & 2 & 3\\ [0.5ex] 
        \hline
        $p_X(x)$ & $0.1$ & $0.1$ & $0.1$ & $0.2$ & $0.2$ & $0.3$ \\ 
        \hline
    \end{tabular}
\end{center}

\begin{itemize}
\item Нам нужно вычислить $E[Y]$, где $Y = \left( X - E[X] \right)^2$. Сразу воспользуемся подходом, позволяющим миновать шаг построения таблицы вероятности для $Y$:
\begin{gather*}
    E[Y] = \sum \limits_{i=1}^{6} g(x_i) P_X(X = x_i) = \sum \limits_{i=1}^{6} (x_i - 1.2)^2 P_X(X = x_i) = \\
    = 1.44 \cdot 0.1 + 4.84 \cdot 0.1 + 10.24 \cdot 0.1 + 0.04 \cdot 0.2 + 0.64 \cdot 0.2 + 3.24 \cdot 0.3 = 2.76
\end{gather*}

\item Но для дисперсии у нас есть ещё более умный подход, описанный на одном из слайдов выше:
\begin{gather*}
    Var[X] = E \left[X^2 \right] - \left( E \left[ X \right] \right)^2 = 4.2 - \left( 1.2 \right)^2 = 2.76 \\
    E \left[ X^2 \right] = \sum \limits_{i=1}^{6} (x_i)^2 P_X(X = x_i) = 0.1 + 4 \cdot 0.1 + 1 \cdot 0.2 + 4 \cdot 0.2 + 9 \cdot 0.3 = 4.2
\end{gather*}

\item \textit{Voilà!}

\end{itemize}

## Функция распределения (кумулятивная/интегральная)

- :::{.callout-definition}
**Функция распределения (кумулятивная/интегральная) (CDF)** случайной величины $X$ определяется как:
$$
    F_X(x) = P(X \leq x), \quad F_X(x): \mathbb{R} \longrightarrow [0,1] \subset \mathbb{R}
$$
Для дискретной случайной величины:
$$
    F_X(x) = \sum_{i: \, x_i \leq x} P_X(X = x_i)
$$
:::

- $F_X(x)$ показывает вероятность того, что случайная величина $X$ примет значение, не превышающее $x$. Иными словами, сколько вероятности "скопилось", аккумулировалось, до значения $x$.

## Свойства CDF для дискретных случайных величин

\begin{enumerate}
    \item \textbf{Монотонность}: $F_X(x)$ неубывающая функция
    
    \item \textbf{Граничные значения}: 
    \begin{itemize}
        \item $\lim\limits_{x \to -\infty} F_X(x) = 0$
        \item $\lim\limits_{x \to +\infty} F_X(x) = 1$
    \end{itemize}
    
    \item \textbf{Ступенчатая функция}: CDF дискретной с.в. имеет скачки в точках, где $p_X(x) > 0$
\end{enumerate}

## Использование CDF для вычисления вероятностей

CDF позволяет легко вычислять вероятности попадания в интервалы:

::: {.callout-tip title="Формулы для вычисления вероятностей"}
\begin{enumerate}
\item $P(X \leq a) = F_X(a)$
\item $P(X < a) = F_X(a^-) = \lim\limits_{x \to a^-} F_X(x)$
\item $P(X > a) = 1 - F_X(a)$
\item $P(X \geq a) = 1 - F_X(a^-)$
\item $P(a < X \leq b) = F_X(b) - F_X(a)$
\item $P(a \leq X \leq b) = F_X(b) - F_X(a^-)$
\end{enumerate}
:::

## Примеры вычислений с CDF

**Для распределения X (на слайдах ранее):**
\begin{enumerate}
    \item $P(X \leq 1) = F_X(1) = 0.5$

    \item $P(X > 2) = 1 - F_X(2) = 1 - 0.7 = 0.3$

    \item $P(-1 < X \leq 2) = F_X(2) - F_X(-1) = 0.7 - 0.2 = 0.5$

    \item $P(0 \leq X < 3) = F_X(3^-) - F_X(0^-) = 0.7 - 0.2 = 0.5$
\end{enumerate}

## Распределение Бернулли

Начнем с простейшего случая - **одного случайного эксперимента** с двумя возможными исходами.

:::{.callout-definition}
**Распределение Бернулли** - это распределение случайной величины $X$, которая принимает только два значения:
$$
X = \begin{cases}
1 & \text{с вероятностью } p \text{ ("успех")} \\
0 & \text{с вероятностью } q = 1-p \text{ ("неудача")}
\end{cases}
$$
Обозначается как $X \sim Bern(p)$, где $p \in [0, 1]$ - параметр распределения.
:::

## Характеристики распределения Бернулли

::: {.callout-tip title="Основные характеристики"}
Для случайной величины $X \sim Bern(p)$:
\begin{itemize}
    \item Математическое ожидание:
    $$E[X] = 0 \cdot (1-p) + 1 \cdot p = p$$

    \item Дисперсия:
    $$Var[X] = E[X^2] - (E[X])^2 = p - p^2 = p(1-p)$$

    \item Стандартное отклонение:
    $$\sigma = \sqrt{p(1-p)}$$
\end{itemize}
:::

## Примеры распределения Бернулли

\begin{enumerate}
    \item \textbf{Подбрасывание монеты}: "Орел" = 1, "Решка" = 0, $p = 0.5$ (для честной монеты)
    
    \item \textbf{Производственный контроль}: "Дефектное изделие" = 1, "Качественное" = 0
    
    \item \textbf{Медицинский тест}: "Положительный результат" = 1, "Отрицательный" = 0
    
    \item \textbf{Маркетинг}: "Покупка товара" = 1, "Отказ от покупки" = 0
\end{enumerate}

## От одного испытания к процессу

\begin{itemize}
    \item Что если мы хотим провести несколько независимых испытаний Бернулли?

    \item Например: подбросить монету 10 раз, проверить 100 изделий на дефекты, провести 50 медицинских тестов.

    \item Это приводит нас к понятию процесса Бернулли - последовательности независимых испытаний Бернулли.
\end{itemize}

## Процесс Бернулли

:::{.callout-definition}
**Процесс Бернулли** - это последовательность независимых одинаково распределенных случайных величин $X_1, X_2, \ldots, X_n$, где каждая $X_i \sim Bern(p)$:
$$
X_i = \begin{cases}
1 & \text{с вероятностью } p \text{ ("успех")} \\
0 & \text{с вероятностью } 1-p \text{ ("неудача")}
\end{cases}
$$
:::


Ключевое отличие от одного испытания: теперь у нас есть **последовательность** результатов.

## Свойства процесса Бернулли

\begin{enumerate}
    \item \textbf{Независимость}: Результат каждого испытания $X_i$ не зависит от результатов других испытаний
    
    \item \textbf{Два исхода}: Каждое испытание может закончиться только успехом (1) или неудачей (0)
    
    \item \textbf{Постоянная вероятность}: Вероятность успеха $p$ не изменяется от испытания к испытанию
\end{enumerate}

## Примеры процесса Бернулли

\begin{enumerate}
    \item \textbf{Серия подбрасываний монеты}: Последовательность $(X_1, X_2, \ldots, X_{10})$ для 10 подбрасываний
    
    \item \textbf{Контроль качества}: Проверка партии из $n$ изделий, где каждое может быть дефектным независимо от других
    
    \item \textbf{Клинические испытания}: Тестирование лекарства на $n$ пациентах, где каждый может показать положительную реакцию
\end{enumerate}

## Биномиальное распределение

- Теперь, когда у нас есть процесс Бернулли, возникает естественный вопрос: **"Сколько успехов мы получим в $n$ испытаниях?"**

:::{.callout-definition}
**Биномиальное распределение** описывает количество успехов в $n$ независимых испытаниях Бернулли с вероятностью успеха $p$.
Если $X_1, X_2, \ldots, X_n$ - процесс Бернулли с параметром $p$, то случайная величина:
$$Y = X_1 + X_2 + \ldots + X_n$$
имеет биномиальное распределение и обозначается $Y \sim Bin(n, p)$.
$Y$ может принимать значения $0, 1, 2, \ldots, n$ (от $0$ до $n$ успехов).
:::


## Функция вероятности биномиального распределения

Вероятность получить ровно $k$ успехов в $n$ испытаниях:

$$
    P(Y = k) = C_n^k p^k (1-p)^{n-k}
$$

::: {.callout-tip title="Интуиция формулы"}
\begin{itemize}
    \item $p^k$ - вероятность $k$ успехов
    \item $(1-p)^{n-k}$ - вероятность $(n-k)$ неудач  
    \item $C_n^k$ - количество способов выбрать $k$ позиций из $n$ для размещения успехов
\end{itemize}
:::

## Характеристики биномиального распределения

::: {.callout-tip title="Основные характеристики"}
Для случайной величины $Y \sim Bin(n, p)$:

**Математическое ожидание:**
$$E[Y] = np$$

**Дисперсия:**
$$Var[Y] = np(1-p)$$

**Стандартное отклонение:**
$$\sigma = \sqrt{np(1-p)}$$
:::

**Вывод математического ожидания:** $E[Y] = E[X_1 + X_2 + \ldots + X_n] = E[X_1] + E[X_2] + \ldots + E[X_n] = p + p + \ldots + p = np$

## Визуализация биномиального распределения

::: {.callout-note title="Наблюдения"}
**p < 0.5** (например, p = 0.3): Распределение смещено влево, большинство значений сосредоточено в области малых k

**p = 0.5**: Распределение симметрично относительно среднего значения

**p > 0.5** (например, p = 0.7): Распределение смещено вправо, большинство значений сосредоточено в области больших k
:::

## Пример: подбрасывание монеты

Подбросим честную монету 10 раз. Какова вероятность получить ровно 6 орлов?

- $n = 10$, $p = 0.5$, $k = 6$, $Y \sim Bin(10, 0.5)$

$$P(Y = 6) = C_{10}^6 (0.5)^6 (0.5)^4 = \frac{10!}{6! \cdot 4!} (0.5)^{10}$$

$$= \frac{10 \cdot 9 \cdot 8 \cdot 7}{4 \cdot 3 \cdot 2 \cdot 1} \cdot \frac{1}{1024} = 210 \cdot \frac{1}{1024} \approx 0.205$$

**Ожидаемое количество орлов:** $E[Y] = 10 \cdot 0.5 = 5$

## Пример: контроль качества

На заводе производят детали, 3% из которых бракованные. Проверяют партию из 100 деталей.

- $n = 100$, $p = 0.03$ (вероятность брака)
- $Y \sim Bin(100, 0.03)$ - количество бракованных деталей

**Характеристики:**
\begin{itemize}
    \item $E[Y] = 100 \cdot 0.03 = 3$ (ожидаем 3 бракованные детали)
    \item $Var[Y] = 100 \cdot 0.03 \cdot 0.97 = 2.91$
    \item $\sigma = \sqrt{2.91} \approx 1.71$
\end{itemize}

**Вероятность не найти ни одной бракованной детали:**
$$P(Y = 0) = C_{100}^0 (0.03)^0 (0.97)^{100} = (0.97)^{100} \approx 0.048$$


## Применения биномиального распределения

\begin{enumerate}
    \item \textbf{Медицина}: Количество пациентов, для которых лекарство окажется эффективным
    
    \item \textbf{Маркетинг}: Количество покупателей, совершивших покупку после просмотра рекламы
    
    \item \textbf{Социология}: Количество респондентов, давших положительный ответ в опросе
    
    \item \textbf{Спорт}: Количество побед команды в серии матчей
\end{enumerate}
